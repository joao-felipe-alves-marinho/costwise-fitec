# CostWise
## Requisitos:
- Node.js


### Crie um arquivo chamado .env 
#### Dentro dele adicione a variavel 
```
VITE_API_URL
``` 
#### Passe a url base da API nessa variavel

### Instale os packages com o comando
``` 
npm install
```
### Para Iniciar o servidor utilize o comando
```
npm run dev
```
